Contact: rowan.gifford@gmail.com
Date: 22th May 2015

2 layer PTH 1.6mm FR4 1 oz Cu I.S.

PCB size: 93.88mm x 43.92mm

Files:

SENSORSTATION_BOTTOM_MASK.gbs - Bottom layer solder mask
SENSORSTATION_BOTTOM_COPPER.gbl - Bottom copper layer
SENSORSTATION_TOP_PASTE.gtp - Top layer solder paste (no paste on bottom layer)
SENSORSTATION_TOP_MASK.gts - Top layer solder mask
SENSORSTATION_TOP_COPPER.gtl - Top copper layer
SENSORSTATION_TOP_SILK.gto - Top layer silkscreen (no silkscreen on bottom layer)
SENSORSTATION_TOP_POS.pos - Top layer pick and place position file. Origin 0,0 is located at fiducial FID1. Zero orientation follows IPC-7351C Level A (Pin 1 in Upper left). No SMD on bottom layer.
SENSORSTATION_DRILL.drl - Drill file, all plated through hole.
SENSORSTATION_BOARD_OUTLINE.gbr - PCB outline, standard 2mm router
SENSORSTATION_TOP_ASSY.pdf - Top assembly file, shows component placement and references for population. Also see sensor_station_PCB_guide.pdf
